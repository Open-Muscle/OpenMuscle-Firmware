# OpenMuscle Firmware

This repository contains the MicroPython firmware for the OpenMuscle devices.

## Devices

- **OpenMuscle Labeler**
- **OpenMuscle Sensor Bracelet**

## Setup Instructions

1. Clone this repository.
2. Install MicroPython on your ESP32 devices.
3. Use `mpremote` to upload the firmware to the devices.

## Directory Structure

- `Labeler/src/`: Firmware source code for the Labeler.
- `SensorBracelet/src/`: Firmware source code for the Sensor Bracelet.
